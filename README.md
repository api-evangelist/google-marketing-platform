# Google Marketing Platform Admin API

The Google Marketing Platform Admin API provides programmatic access to manage links between Google Marketing Platform organizations and Google Analytics accounts. It enables creating, updating, deleting, and listing organization links and managing service levels for integrated marketing analytics.

## APIs

- **Google Marketing Platform Admin API** - Programmatic management of organization-level settings, Analytics account links, and service level configuration.

## Artifacts

- [APIs.yml](apis.yml) - API discovery document following the APIs.json specification.
- [OpenAPI](openapi/openapi.yml) - OpenAPI 3.1.0 specification for the Marketing Platform Admin API.
- [JSON Schema](json-schema/json-schema.yml) - JSON Schema (draft 2020-12) for API resource models.
